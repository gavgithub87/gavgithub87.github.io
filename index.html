Welcome to the documentation about the SOS GmbH repositories

This is a short documentation about:

  • what projects the repositories are about
  • what repositories are needed in order to compile the sources
  • where the instructions for compiling are located

Projects

The projects the repositories correspond to are JobScheduler and JADE. JobScheduler and JADE are the two Open Source products the company offers.

Needed repositories

As you can see on our GitHub site, there are twelve repositories available. Only three of them are relevant for you when building the sources:

  • scheduler-engine-doc
  • scheduler-engine
  • all
The rest of the repositories are submodules from the repository all which had to be published in order to be able to be referenced when building this repository.

Build instructions and Support

The build instructions are documented in our Product Knowledge Base, you can find them at https://kb.sos-berlin.com/x/dYBB. Having troubles with the Build? Have a look at the chapter Support in this article. We'll help you to sort it out.
